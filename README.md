## Momo
简单的图片浏览服务

### 使用
```bash
momo.exe [OPTIONS]
```
```bash
Options:
  -d, --directory <directory>  设置工作目录
  -p, --port <port>            监听的端口号 [default: 3000]
  -l, --loglevel...            启用调试信息
  -w, --width <WIDTH>          设置图片的宽度，该设置决定了瀑布流的宽度 [default: 350]
  -h, --help                   Print help
  -V, --version                Print version
```
